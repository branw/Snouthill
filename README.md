# Snouthill

Snouthill is a server emulator for old versions of Habbo Hotel.

It is currently in the very early stages of development and targets v14, which
was released between May and July of 2007.

## Progress

- [x] Login
- [x] Registration

- [ ] Public rooms
- [ ] Private rooms

- [ ] Bots
- [ ] Infobus

- [ ] Messenger

- [ ] Catalog
- [ ] Furni recycler
- [ ] Trading

- [ ] Trax
- [ ] Rollers

- [ ] Game Hall

- [ ] Moderation
